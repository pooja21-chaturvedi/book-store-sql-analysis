
# Book Store SQL Analysis Project

## Project Overview

This project analyzes bookstore sales data using PostgreSQL. The data was imported from CSV files and stored in relational database tables. SQL queries were used to answer business questions related to sales, customers, books, and inventory.

## Dataset

* Books.csv
* Customers.csv
* Orders.csv

## Tools Used

* PostgreSQL
* SQL
* CSV Import/Export

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* INNER JOIN
* Aggregate Functions (SUM, AVG, COUNT)
* Primary Keys
* Foreign Keys

## Business Questions Solved

1. Which books generated the highest sales?
2. Who are the top customers by spending?
3. Which genre is the most popular?
4. Which books have the lowest stock?
5. What is the total revenue generated?

## Skills Demonstrated

* SQL Querying
* Data Analysis
* Database Design
* Data Import and Export
* Relational Database Management

## Project Outcome

Successfully analyzed bookstore data and generated insights on customer behavior, sales performance, and inventory management using PostgreSQL.
